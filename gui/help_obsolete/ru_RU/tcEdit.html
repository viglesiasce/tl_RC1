Редактирование тестов

Назначение:

Раздел "Описание тестов" – это место, где пользователь может просмотреть и отредактировать компоненты, категории тестов,а также описания тестовых примеров, относящихся к заданному Тестовый проект. Пользователь может просмотреть различные версии теста.

Что делать:

  1. Выберите название Тестового проекта из выпадающего меню, расположенного в правом верхнем углу экрана.
  2. Создайте новый компонент, категорию или тест для этого продукта. (Вы можете сменить тестовый проект в правом верхнем углу).
  3. Используя древовидную структуру панели навигации слева, отредактируйте описания тестовых примеров.
  4. 4. Когда тест будет готов, включите его в План тестирования.

Все тестовые примеры в TestLink’е распределяются по N уровням тестовых наборов. Вы можете описать содержание тестовых наборов. Эта информация может быть распечатана вместе с описанием самих тестов.
