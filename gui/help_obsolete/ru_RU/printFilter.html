Печать тестовых примеров: выбор опций

Закрыть Назад Оглавление

В этом разделе пользователь может установить параметры отображения перечня тестов в выводимом документе. Выбор данных осуществляется установкой флажка на соответствующей опции и нажатием кнопки "Установить". Для вывода информации о тестах необходимо нажать на соответствующую категорию (компонент) в окне структуры тестов.

Показать заголовок документа

При выборе этой опции вместе с набором тестов выводится информация из полей описания Компонентов и Категорий: "Введение", "Обзор", "Ссылки", "Методология", "Ограничения".

Показать содержание тестового примера

При выборе этой опции выводится вся информация о тестах, включая следующие поля: "Описание теста", "Шаги", "Ожидаемая реакция" и "Ключевые слова".

Поле "Описание теста"

При выборе этой опции выводится информация о тестах, включающая только название и описание теста. Если для вывода информации о тестах пользователь включил опцию "Показать содержание тестового примера", а данную опция отключил, то поле "Описание теста" все равно будет выведено для печати.

Показать оглавление документа

При выборе этой опции в выходном документе автоматически формируется оглавление, составленное из внутренних гиперссылок.

Формат вывода

Пользователь может вывести информацию о тестах в двух доступных форматах: HTML и MS Word.
