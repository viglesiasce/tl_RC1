Обзор

Для быстрой навигации по программе в Teslink предусмотрено использование "горячих" клавиш (ALT + <клавиша>).

Список доступных клавиш:

  • [h] Домой
  • [i] Мои данные
  • [q] Выход
  • [u] Персонал (только для администратора)

Поддержка браузеров

Использование "горячих" клавиш поддерживается следующими web-браузерами:

  • Internet Explorer 4 (Windows) 5+ (Mac)
  • Mozilla (Windows + Linux)
  • Netscape 6.1+ (Windows)
  • Opera 7 (Windows + Linux)
