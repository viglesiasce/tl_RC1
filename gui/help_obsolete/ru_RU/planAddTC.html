Добавить тестовый пример в тест-план

Назначение:

Этот раздел предназначен для того, чтобы пользователь с правами не ниже руководителя (leader) мог добавлять тесты в план тестирования.

Что делать:

  1. Выберите компонент для просмотра всех его категорий и тестов, входящих в них. Нажмите на категорию и просмотрите все тесты этой категории.
  2. Пометьте флажками те тесты, которые вы хотите добавить в План тестирования.
  3. Нажатие кнопки "Добавить выбранные тесты" перенесет тесты в текущий план, при этом Testlink проверяет, чтобы каждый тест не мог быть добавлен многократно.
  4. Вы можете также отобрать тесты по ключевому слову.

см. также: Справка TestLink
