Назначение

Данная таблица дает возможность пользователю отобрать тесты для прогона по следующим параметрам.

Принадлежность

Пользователь может отбирать тесты по их владельцу. Принадлежность определяется на уровне категорий, ее задает руководитель (leader), и она может быть изменена на странице Установка параметров для группы тестов.

Ключевое слово

Пользователи могут делать выборку тестов по ключевому слову. Ключевые слова задаются на странице Создания/ редактирования/ удаления тестов или на странице "Задать ключевые слова для теста". Создание, редактирование и удаление и Ключевых слов является правом руководителя. Задавать ключевые слова для тестов может и пользователь с правами тестера.

Версия (сборка)

Пользователи могут отбирать тесты для прогона по версии (сборке) программного обеспечения. Версии являются основным компонентом для прослеживаемости тестов. Тест может быть запущен только один раз на одной сборке.
Версии (сборки)создаются руководителем на странице "Создать новую версию (сборку)".

Результат прогона

Пользователи могут отбирать тесты по результатам прогона на определенной версии (сборке). Тесты может быть пройденным, проваленным, блокированным или невыполненным.

Последний результат прогона

По умолчанию или если не выбрано значение "Последний прогон", дерево тестов будет отсортировано по выбранной версии. При этом в структуре тестов цветом будет отображаться их статус.
Например: На навигационной панели пользователь назначил отбор тестов по версии 2, и не выбрал значение "последний прогон". В результате будут показаны все тесты со своими статусами из версии 2. Так, если тест 1 пройден в версии 2, то он будет отображаться зеленым цветом.
Если пользователь решил задать значение "последний прогон" тесты в дереве будут окрашено согласно результатам последнего прогона.
Например: Пользователь назначил отбор тестов по версии 2, и на этот раз выбрал значение "последний прогон". Все тесты будут окрашены согласно результату последнего прогона. Поэтому, если тест 1 оказался пройденным только в версии 3, он будет выделен зеленым цветом, несмотря на то, что пользователь выбрал версию 2.
