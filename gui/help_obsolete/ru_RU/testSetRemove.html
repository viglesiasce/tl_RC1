Удаление теста из плана тестирования

Назначение:

На этой странице пользователь может сделать тесты неактивными. Неактивные тесты не будут показаны на странице прогона тестов.

Примечание: Результаты более раннего прогона уже неактивного теста будут присутствовать в отчетах.

Что делать:

  1. Выберите категорию для просмотра всех входящих в нее тестов.
  2. Установите флажки для тестов, которые будут удалены из плана.
  3. Нажмите кнопку "Удалить...".
