Страница поиска тестов

Назначение:

Для поиска используются ключевые слова и строки поиска по всем полям описания тестового примера. Поиск не чувствителен к регистру. Результат поиска включает тесты только активного продукта.

Что делать:

  1. Впишите строку поиска в нужное поле. Неиспользованные поля оставьте пустыми.
  2. Выберите требуемое ключевое слово, или оставьте значение "не применимо".
  3. Нажмите кнопку "Искать".
  4. Все тесты, соответствующие строкам поиска, показаны. Вы можете модифицировать выбранные тесты, используя кнопку "Редактировать", расположенную под строкой заголовка теста.
