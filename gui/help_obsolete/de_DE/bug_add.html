Hinzufuegen von Bugs zu einem Testfall

TestLink hat einen sehr einfachen Integrationsmechanismus fuer ein Bug-Tracking-System (BTS).
Es kann aber weder direkt ein Bug in diesem erstellen, noch die ID aus einem BTS fuer einen Bug zurueckbekommen.
Die Integration wird daher einfach durch Links zum BTS hergestellt, wodurch folgende Moeglichkeiten angeboten werden:

  • Einen neuen Bug erfassen.
  • Informationen zu einem vorhandenen Bug anzeigen.

Anleitung, um einen Bug hinzuzufuegen:

  • Schritt 1: Das BTS ueber den Link oeffnen, um einen neuen Bug zu erfassen.
  • Schritt 2: Die Bug-ID, welche vom BTS vergeben wurde, merken..
  • Schritt 3: Die Bug-ID in das entsprechende Feld in TestLink schreiben.
  • Schritt 4: Den Button "Bug hinzufuegen" druecken.
Nach dem Hinzufuegen des Bugs und dem Schliessen dieser Seite kann man die relevanten Informationen des Bugs auf der Hauptseite sehen.
