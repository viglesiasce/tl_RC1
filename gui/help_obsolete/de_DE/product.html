Scope

This is the main point for data division. Choose a product in a top right corner menu.

Features

Test Specification

This section is where users create, edit and delete components, categories and test cases. There is also a page that allows users to search through all of the test cases.
Users with the proper permissions can also change the order in which test cases and categories appear when listed; and well as import CSVs.

Keywords

TestLink enables users to filter test cases by their components, categories and by user specified keywords. The keyword management section is where users with the proper permissions can add, edit, and delete these keywords.
Note: Keywords can be assigned in the test case management section as well as the batch keyword assignment page.

Product Management

The product management section is primarily an admin function. It allows users to create and edit products. Each product can have different settings and available features.
