Testfallausfuehrung

Ziel:

Erlaubt einem Benutzer einen Testfall auszufuehren. Der Benutzer kann dabei ein Testresultat einem Testfall gegen einen Build zuweisen. Siehe Hilfe fuer mehr Informationen ueber Filter und Einstellungen.

Anleitung:

  1. Der Benutzer muss einen Build fuer den Testplan erstellt haben.
  2. Waehle einen Build in der Auswahlbox aus und druecke den Aktualisieren-Button im Navigationsbereich.
  3. Waehle eine Testsuite, um alle darunterhaengenden Testsuiten und alle Testfaelle zu sehen.
  4. Fuelle das Testresultat aus und ergaenze passende Anmerkungen oder Bugs.
